# StoreOS PWA Installation Guide

## How to Add StoreOS to Your Home Screen

### 📱 iPhone/iPad (Safari Required)

1. **Open in Safari**
   - Must use Safari browser (not Chrome or other browsers)
   - Go to your StoreOS URL

2. **Tap Share Button**
   - Look for the share icon (square with arrow) at the bottom of Safari
   - Usually located next to the address bar

3. **Find "Add to Home Screen"**
   - Scroll down in the share menu
   - Tap "Add to Home Screen" option

4. **Confirm Installation**
   - App name will show as "StoreOS"
   - Icon will appear as teal box logo
   - Tap "Add" button

5. **Launch App**
   - StoreOS icon now appears on your home screen
   - Tap to open - runs fullscreen with no browser bar
   - You're already logged in!

### 🤖 Android (Chrome Recommended)

1. **Open in Chrome**
   - Go to your StoreOS URL in Chrome browser

2. **Tap Menu Button**
   - Look for three dots (⋮) in top-right corner
   - Tap to open menu

3. **Install App**
   - Look for "Install app" or "Add to Home screen" option
   - Tap to install

4. **Confirm Installation**
   - Review app info and tap "Install"
   - App appears on your home screen

### 💻 Desktop (Chrome/Edge)

1. **Open in Supported Browser**
   - Chrome, Edge, or Opera required
   - Go to your StoreOS URL

2. **Look for Install Icon**
   - Check address bar for install icon (⬇️)
   - Or click menu (⋮) → "Install StoreOS"

3. **Install Application**
   - Click install button
   - Confirm installation
   - App appears in applications/start menu

## 🔧 Troubleshooting

### "Add to Home Screen" not showing?
- **iPhone**: Ensure you're using Safari, not Chrome
- **Android**: Make sure you're using Chrome
- **Desktop**: Try refreshing the page first

### App not working offline?
- First-time installation requires internet
- Try opening the app once while online
- Service worker will cache everything automatically

### Login required each time?
- This is normal behavior for security
- Your session persists but may expire after inactivity
- Just log in once and you'll stay logged in
